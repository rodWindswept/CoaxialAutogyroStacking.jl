# Issue Tracker: GitHub

Issues for this repository are tracked on GitHub.

## Commands

- **Create issue**: `gh issue create --title "<title>" --body "<body>" --label "<label>"`
- **List issues**: `gh issue list`
- **View issue**: `gh issue view <number>`
- **Edit issue**: `gh issue edit <number> --add-label "<label>" --remove-label "<label>"`

## Workflow

Skills like `to-issues` and `to-prd` will use the `gh` CLI to interact with GitHub Issues.
